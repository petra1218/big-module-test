2.1kafka接图
基本信息
Path： /receive
Method： HEAD
接口描述：

1. Kafka Topic 名称
项 值
Topic 名称 Q_VEHICLE_FRONT_FACE_SHARE_TO_XQ(支持自定义)
说明 第三方系统需向该 Topic 获取数据
2. 推送数据字段说明
DeviceID 与 IndexCode 为二选一的必要参数，优先使用 DeviceID；若 DeviceID 为空但
IndexCode 有值，则将 IndexCode 赋值给 DeviceID，统一设备标识维度
字段名 类型 说明 其他
ID string 数据 ID 32 位
DeviceID string 设备 ID 二选一的必要参数
IndexCode string 对接设备唯一标识 二选一的必要参数
DeviceName string 设备名称 必填
SceneImageUrl string 图片路径 必填
CatTime string 抓拍时间 必填，格式 yyyy-MM-dd HH:mm:ss
Lat DOUBLE 纬度
Lng DOUBLE 经度
Expand string 扩展
3. 推送 JSON 数据示例
{
"ID": "3066b46cd2a152093ec29354827d64d",
"DeviceID": "deviceId001",
"IndexCode": "indexcode001",
"DeviceName": "设备名称",
"SceneImageUrl": "http://ip:port/20250101/image.jpg",
"CatTime": "2026-03-01 18:18:18",
"Lat": 116.40,
"Lng": 39.91,
"Expand": "{\"code\":\"asdcfsadf123\",\"remark\":\"remark\"}"
}

4.1 Expand 结构说明

{
